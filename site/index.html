My coding journey

Welcome

My name is Tom, I am a service engineer by trade working on professional audio equipment but I have always been interested in coding. This project is designed to be an example of my work and will change and progress as I learn more. I started learning HTML, CSS and JavaScript using the FreeCodeCamp curriculum in late November 2025 with a very basic understanding of HTML and no experience with CSS or JavaScript.

Projects

Below are links to the projects I have created using HTML only

Build a Curriculum Outline

This is the very first project I built, it is a very basic static webpage only utilising the <h1>, <h2> and <p> elements of HTML

As you can see, this project was very simple but I found it to be an enjoyable first task in the course. I like that I was able to see code that I had written working, it gave me a sense of achievment and gave me a rush of motivation.

Build a cat photo app

The second project was to build a simple cat photo app. This was the first introduction to adding images to a page using the <a> anchor element, making clicked links open in a new tab, adding sources via the src attribute and working with the alt attribute for accesibility purposes. I was also introduced to the use of the <ul> (unordered, bulleted list), <ol> (ordered, numbered list) and <li> (list item) elements. The <figure> and <figcaption> elements were used in this project to caption pictures on the page.

Build a Bookstore page

This project introduced the <button> element used to create buttons on a page, combined with the anchor (<a>) element and it's attributes target and href to link it to a resource. The buttons in this project don't actually link to anything but it was good to learn how to set them up. The <div> element was also introduced at this point, it doesn't do anything until it is styled using CSS but it can be used to divide your content.

Build an audio and video player

This project introduced the <audio> and <video> elements used with various attributes, in this case width which determines the width of the video player on the page, controls which allows the user to control the video or audio player and loop which, in this case, loops the audio file

Build a heart icon

In this project I worked with the <svg> element and the attributes width and height to determine the width and the height of the icon but also viewbox to set the size of the viewbox where the icon will be displayed, if this is set to be smaller than the icon then part or all of the icon will not show up. I also used the attribute fill to fill the icon with colour, in this case red.See the source code which shows the path that is drawn to create the icon.
